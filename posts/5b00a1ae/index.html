[VS Code] VS Code Plugins

Posted on 2018-01-22 In VS Code Views: Disqus:
3.6k 3 mins.

最近從 Sublime 跳槽到 Visual Studio Code(VS Code)，雖然在開啟速度上，會明顯地感覺到比 Sublime 慢，但是 VS Code 有很多的 plugin 可以讓開發效率更好，雖然 Sublime 也有不少 plugin，但是有些很基本的功能還要另外安裝，有點麻煩(例如: SideBarEnhancements，這大概是裝完 Sublime 之後第一個裝的 plugin XD)，再加上常常在儲存檔案時出現提醒視窗也很煩，最後就決定跳槽到 VS Code，在這裡紀錄一下自己在開發時常用的 plugins.

外觀

Flatland Monokai Theme

  • 更多主題: Themes

Material Icon Theme

File icon

Folder icon

Linter

ESLint

Integrates ESLint into VS Code.

  • 常見的規則有Airbnb、Google

stylelint

Linter for CSS, SCSS, Less.

markdownlint

Linter for Markdown.

Formatter

Beautify

程式碼自動排版，支援 JavaScript, JSON, CSS, Sass 和 HTML.

  • Ctrl+Shift+P → Beautify file
  • 自訂快捷鍵: Alt+C B

Snippets

HTML snippets

This extension adds rich language support for the HTML Markup to VS Code, including: Full HTML5 Tags, Colorization and Snippets.

JavaScript (ES6) code snippets

This extension contains code snippets for JavaScript in ES6 syntax for Vs Code editor (supports both JavaScript and TypeScript).

jQuery Code Snippets

Over 130 jQuery Code Snippets for JavaScript code.

Vue 2 Snippets

This extension adds Vue 2 Code Snippets into Visual Studio Code.

Auto-completes

Auto Rename Tag

自動 rename 對應的 tag.

npm Intellisense

Autocompletes npm modules in import statements.

Path Intellisense

Autocompletes file path.

Other useful plugins

Dash

方便快速查找 API 資料。

Vetur

Vue tooling for VS Code, powered by vue-language-server.

Auto-Open Markdown Preview

Preview markdown file.

Markdown All in One

All you need for Markdown (keyboard shortcuts, table of contents, auto preview and more).

Markdown Preview Enhanced

Preview markdown file, 且支援許多好用的功能，例如: 插入/上傳圖片、將 markdown file 轉為 PDF 等功能。

Color Picker

調色盤，快速選取顏色:

  • 快捷鍵: Ctrl+C P
  • 如果無法顯示調色盤，須修改 VS code config (settings.json):

Debugger for chrome

Debug JavaScript code in Chrome browser.

.ejs

支援 EJS.

Easy LESS

在儲存時，自動將 Less 轉為 css.

Git History(git log)

查看 Git history, git log.

npm

支援 npm.

open in browser

在 browser 中開啟目前的檔案:

  • Alt+B: Open in default browser
  • Alt+Shift+B: Open in browser

Python

  • Linting
  • Intellisense
  • Auto indenting
  • Code formatting
  • Code refactoring
  • Viewing references, code navigation, view signature
  • Excellent debugging support
  • Running and debugging Unit tests
  • Execute file or code in a python terminal
  • Local help file
  • Snippets

Regex Previewer

Preview Regex 比對結果。

SFTP

SFTP/FTP 同步。

vscode-hexo

Provides Hexo init, new, generate, server, deploy, publish, clean commands in the VSCode Editor.

Code Spell Checker

自動檢查拼字是否錯誤。

VSCode Reveal

使用 VSCode 做 slides.

Swagger Viewer

方便直接瀏覽 Swagger API 文件。

Settings Sync

透過 Gist 同步 VS code settings, 不需要在每台電腦都重新設定一次 VS Code 的環境。
